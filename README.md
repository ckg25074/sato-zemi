# タイトル1「太宰府の観光名所を紹介するWebページ」
## 概要
### 目的・背景
- 情報提供やガイド目的で大宰府に行きたいと思わせるようなものを作りたい。
- 今まで来てくれた観光客はどういうところを楽しんでくれたのか、実際に自分が行って確かめてみたいため。

### ページ構成

#### ページ１　「HOME」
#### ページ２　「概要」
#### ページ３　「観光スポットランキング」
#### ページ４　「太宰府市マップ」
#### ページ５　「グルメ」

### ページの内容について
- ページ1「HOME：ホームページの最初の欄　ここに太宰府市の色々な写真を張る　他のページに誘導するための軽い宣伝をする 文章等は全て自分で考えまとめて書く」　　
- ページ2「概要：太宰府市全体の紹介や歴史の解説」
- ページ3「観光スポット紹介：太宰府市の人気な観光スポットを写真付きで紹介」
- ページ4「太宰府市マップ：太宰府市の上からのマップを上記の観光スポットと照らし合わせて表示する」
- ページ5「グルメ：太宰府市で有名なご当地グルメを(一部撮れない可能性あり)写真付きで紹介」

### 備品・準備について
- 太宰府市に行くための交通費、ホームページなどの資料集め
- 大宰府周辺の写真(許可をとれた場所のみ)
- 写真は人の顔を写さないこと
- 可能であればグルメなども調べる
- 写真枚数10枚ほど撮る(使う枚数は8枚で予備に2枚撮る)
- 加工は上手く見せられるように工夫する

# タイトル2「天神の紹介Webページ」
## 概要
### 目的・背景
- 福岡の大規模な街の一つである天神　自分も行く機会が少なくないので知らないことやどういうところが人気なのかを知るため
- 天神駅以外での魅力を知るため

### ページ構成

#### ページ１　「HOME」
#### ページ２　「概要」
#### ページ３　「観光スポットランキング」
#### ページ４　「天神マップ」

### ページの内容について
- ページ1「HOME：ホームページの最初の欄」　　
- ページ2「概要：全体の紹介や天神駅について紹介」
- ページ3「観光スポット紹介：天神の人気な観光スポットを写真付きで紹介」
- ページ4「天神マップ：太宰府市の上からのマップを上記の観光スポットと照らし合わせて表示する」

### 備品・準備について
- 天神の情報を集めるために色々なホームページや身内に聞き込みをする
