# タイトル1「太宰府天満宮を紹介するWebページ」

# テーマ
### 「自然と学問の神社　大宰府天満宮の良さを知ろう」
# 概要
## ターゲット
- これから太宰府天満宮を訪れる観光客向け

## 背景
- 初めて太宰府天満宮に行った時にとても感動して、街並みも綺麗な和風で自分の好きな場所になったからです。

## 目的
- 自分の好きな場所になった太宰府天満宮をこれから来る観光客の人たちにも知ってほしい、感動してほしいと思い、いろいろな人が見られるホームページを作ることにしました。

- ホームページにしようと考えたのは実際に行った人感想でホームページ内でしか手に入らない情報が手に入り、作った自分も人に見て役立ってくれるならうれしいのでお互いにいい思いが出来るからです。

## 分析
太宰府天満宮は、学問の神様として全国にその名を馳せる神社で毎年多くの受験生やその家族が訪れるだけでなく、美しい自然に囲まれた境内は観光客にも大変人気があり、クチコミやインスタでも学力向上や合格祈願で訪れる人も少なくなく、ご利益ごとにお守りがあるので参拝の時に購入する人もいます


# ページの内容について
- ページ1「HOME：ホームページの最初の欄　ここに太宰府天満宮周辺の写真を張り、他のページを見たくなるような写真を大きく張る」

- ページ2「概要：太宰府天満宮の人気な観光スポットを写真付きで紹介」

- ページ3「歴史：太宰府天満宮の誕生から現在までの歴史を解説(現在調べ中)」

- ページ4「グルメ：太宰府天満宮で有名なご当地グルメを写真付きで紹介」


# 今後の流れ
- ７月：テーマを固めて、どんなWebページを作るかを決める

- ８月：大宰府天満宮周辺の写真を撮る素材集め(写真を撮る許可をもらえたところのみ)

- ９月～１０月：集めた素材を元にWebページのHTML・CSSを打ち込んでいく

- １１月：動作確認

- １２月：問題がなければ完成




# タイトル2「軽自動車の良さをまとめたWebページ」
# テーマ
### 「軽自動車の便利をさらに詳しく」
# 概要
## ターゲット
- これから車を買いたいと思っている人
- 車に興味がある人

## 目的
- 軽自動車の便利さをこれから車を買う予定の人や興味がある人に知ってもらいたい
- ホームページに自分が持っている情報を書いてその人に見てもらい、軽自動車について興味を持ってほしいからです

## 背景
- 自分はよく軽自動車を乗っていて普通車よりも小回りが利いていて便利で、これから車を買う人には軽自動車を乗ってもらいたいなと思ったからです。

## 分析
軽自動車を買ったネットの人の感想としては「税金や保険料などの維持費が安い」「狭い道や駐車場での小回りが利く」などの意見が多く、たくさんの人から使われています。

一方で「長距離走行は不向き」「高速道路でのパワー不足」という意見もあり、自分に合った車選びも必要です。

## ページの内容について
- ページ1「HOME：ホームページの最初の欄　おすすめのポイントを3つ紹介」
　
- ページ2「主な特徴：軽自動車の長所と短所を紹介」

- ページ3「比較：他のサイズの車との違いを解説」

- ページ4「検討中」

（車の会社名や名前は出さない）

## 今後の流れ
- ７月：テーマを決めて、ページ4の中身を考える、Webページの構想を考える　

- ８月：フリー素材を探して、自由に使うことが出来る素材のみ使う

- ９月～１０月：集めた素材を元にWebページのHTML・CSSを打ち込んでいく

- １１月：動作確認

- １２月：問題がなければ完成
