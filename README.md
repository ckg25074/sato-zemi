# タイトル1「太宰府市の観光名所を紹介するWebページ」
## 概要
### 目的・背景
- 情報提供やガイド目的で大宰府に行きたいと思わせるようなものを作りたい。
- 今まで来てくれた観光客はどういうところを楽しんでくれたのか、実際に自分が行って確かめてみたいため。

### ページ構成

#### ページ１　「HOME」
#### ページ２　「概要」
#### ページ３　「観光スポットランキング」
#### ページ４　「グルメ」

### ページの内容について
- ページ1「HOME：ホームページの最初の欄　ここに太宰府市の色々な写真を張る　他のページに誘導するための軽い宣伝をする 文章等は全て自分で考えまとめて書く」　　
- ページ2「概要：太宰府市全体の紹介や歴史の解説」
- ページ3「観光スポット紹介：太宰府市の人気な観光スポットを写真付きで紹介」
- ページ4「グルメ：太宰府市で有名なご当地グルメを(一部撮れない可能性あり)写真付きで紹介」

### 今後の流れと準備
- 太宰府市に行くための交通費、ホームページなどの資料集め
- 大宰府周辺の写真(許可をとれた場所のみ)
- 写真は人の顔を写さないこと
- 可能であればグルメなども調べる
- 写真枚数10枚ほど撮る(使う枚数は8枚で予備に2枚撮る)
- 加工は上手く見せられるように工夫する

# タイトル2「軽自動車の紹介Webページ」
## 概要
### 目的・背景
- 最近自分も免許を取ってよく軽自動車を運転するのでどのようなものがあるか　などいろいろな軽自動車を調べると同時に自分もいつか自分で買うときに参考にしたいと思ったからです

### ページ構成

#### ページ１　「HOME」
#### ページ２　「概要」
#### ページ３　「おすすめ」
#### ページ４　「軽自動車の魅力」

### ページの内容について
- ページ1「HOME：ホームページの最初の欄」　　
- ページ2「概要：軽自動車の種類一覧」
- ページ3「おすすめ：初心者でもベテランの人でも乗りやすく好評なものを紹介」
- ページ4「軽自動車の魅力：軽自動車の他にはない魅力や強みを解説」

### 今後の流れと準備
- まずは単純に車の知識を深める
- 著作物や素材に何を使っていいかなどかはしっかり考えること
- 軽自動車を買った人に買ってよかったことを尋ねる
