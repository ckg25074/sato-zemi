# タイトル1「太宰府の観光名所を紹介するWebページ」
## 概要
### 目的・背景
- 情報提供やガイド目的で大宰府に行きたいと思わせるようなものを作りたい。
- 今まで来てくれた観光客はどういうところを楽しんでくれたのか、実際に自分が行って確かめてみたいため。

### 制作スケジュール
- 7月　Webページの構造を考える
- 8月 実際に大宰府に行って写真撮影等素材集め
- 9月　HTMLの構造や仕組みをちゃんと理解
- 10月 コードの打ち込み　試行錯誤を繰り返す
- 11月　Webページの制作開始
- 12月 Webページの制作～写真や素材の貼り付け
- 1月 構造等の確認
- 2月 1月と同様　完成まで
- 3月 ゼミ発表

### 備品・準備について
- 太宰府市に行くための交通費、ホームページなどの資料集め
- 大宰府周辺の写真(許可をとれた場所のみ)　写真は人の顔を写さないこと
- 可能であればグルメなども調べる
