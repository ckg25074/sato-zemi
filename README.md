# タイトル1「太宰府天満宮を紹介するWebページ」

# テーマ
### 「自然と学問の神社　大宰府天満宮の良さを知ろう」

# 作品概要
- 太宰府天満宮の良さをWebページに掲載して、興味を持ってくれた人に向けて制作したいと思ったからです。

## ターゲット
- 太宰府天満宮に興味がある人達

## 背景
- 初めて太宰府天満宮に行った時にとても感動して、街並みも綺麗な和風で自分の好きな場所になったからです。

## 目的
- 自分の好きな場所である太宰府天満宮をこれから来る観光客の人たちにも知ってほしいと思い、実際に行きたい人が見られるホームページを作ることにしました。

## 分析
- 歴史の始まりとして、太宰府天満宮は、約1,100年以上前に建てられた神社です。
菅原道真という学問に優れた人物が、都から太宰府へ送られ、そこで亡くなりました。道真を敬い、その場所に神社を建てたのが太宰府天満宮の始まりです。
- 今では、道真が「学問の神様」として知られているため、勉学の成功を願う人が多く訪れる神社になり、全国にその名を馳せるようにまりました。そしてご利益ごとにお守りがあるので参拝の時に購入する人もいます


# ページの内容について
- ページ1「HOME：ホームページの最初の欄　太宰府天満宮の人気な観光スポットを写真付きで紹介」

- ページ2「歴史：太宰府天満宮の誕生から現在までの歴史を解説」

- ページ3「グッズ：太宰府天満宮で売られているお守りやペンダントを写真付きで紹介します」

- ページ4「グルメ：太宰府天満宮で有名なご当地グルメを写真付きで紹介します」


# 今後の流れ
- ７月：テーマを固めて、どんなWebページを作るかを決める

- ８月：大宰府天満宮周辺の写真を撮る素材集め

- ９月～１０月：集めた素材を元にWebページのHTML・CSSを打ち込んでいく

- １１月：動作確認

- １２月：問題がなければ完成




# タイトル2「軽自動車の良さをまとめたWebページ」
# テーマ
### 「軽自動車の便利をさらに詳しく」
# 作品概要
- 車を買う人が軽自動車を買いたくなるような便利なWebページ作り

## ターゲット
- これから車を買いたいと思っている人（初心者でもベテランドライバーどちらも）
## 目的
- 軽自動車の便利さをこれから車を買う予定の人に知ってもらいたい
- ホームページに自分が持っている情報を書いてその人に見てもらい、軽自動車について興味を持ってほしいからです

## 背景
- 自分はよく軽自動車を乗っていて普通車よりも小回りが利いていて便利で、これから車を買う人には軽自動車を乗ってもらいたいなと思ったからです。

## 分析
軽自動車を買ったネットの人の感想としては「税金や保険料などの維持費が安い」「小回りが利くので初心者でも乗りやすい」などの意見が多く、初心者から上級者まで幅広く使われています。

一方で「長距離走行は不向き」「高速道路でのパワー不足」という意見もあり、自分に合った車選びも必要です。

## ページの内容について
- ページ1「HOME：ホームページの最初の欄　おすすめのポイントを3つ紹介」
　
- ページ2「主な特徴：軽自動車の長所と短所を紹介」

- ページ3「比較：他のサイズの車との違いを解説」

- ページ4「検討中」

（車の会社名や名前は出さない）

## 今後の流れ
- ７月：テーマを決めて、ページ4の中身を考える、Webページの構想を考える　

- ８月：フリー素材を探して、自由に使うことが出来る素材のみ使う

- ９月～１０月：集めた素材を元にWebページのHTML・CSSを打ち込んでいく

- １１月：動作確認

- １２月：問題がなければ完成
